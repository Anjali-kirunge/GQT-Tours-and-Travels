# ✈️ GQT Tours and Travels Website

This project is a **responsive Tours and Travels website UI** built using HTML, CSS, Bootstrap, and Font Awesome. It showcases travel destinations, booking interface, and a modern travel website layout.

## 🚀 Features
- Responsive navigation bar with icons
- Scrolling announcement banner 📢
- Bootstrap image carousel (slider)
- Destination search section 🔍
- Popular travel destinations cards 🏔️
- Social media integration
- Photo gallery section 📸
- Contact & booking form
- Embedded Google Maps location 📍
- Clean and modern UI design

## 🛠️ Tech Stack
- HTML5
- CSS3
- Bootstrap 5
- Font Awesome Icons

## 📂 Project Structure
- `index.html` – Main webpage
- `style.css` – Custom styling
- External image links for visuals

## 🎯 Key Highlights
- Fully responsive layout
- Uses Bootstrap components (Navbar, Carousel, Cards, Grid)
- Interactive UI without backend
- Beginner-friendly frontend project
- Real-world travel website design

## 📌 How to Run
1. Clone or download the repository
2. Open `index.html` in your browser
